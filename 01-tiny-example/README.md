# Kepler.gl - Múltiples Opciones

## Opción 1: Script automático
```
instalar.bat
```

## Opción 2: Manual paso a paso
```
pip install pandas jupyter
pip install --only-binary=all keplergl
```

## Opción 3: Usar Kepler.gl web (MÁS FÁCIL)
```
python crear_geojson.py
```
Luego ve a https://kepler.gl/demo y arrastra `datos.geojson`

## Opción 4: Versión antigua
```
pip install keplergl==0.2.2
```

## Ejecutar notebook:
```
jupyter notebook kepler.ipynb
```
